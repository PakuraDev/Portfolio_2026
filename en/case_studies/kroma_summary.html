Kroma is the start page that has the honor of being the first page of Firefox I see every day. While it's not the most remarkable thing in the world in terms of design and code (nor is it a very interesting project in general — or maybe I'm just bored of seeing it every day), it's something worth including in my portfolio.

One extra consideration: if you want to use it as your start page, you'll need to install an extension for it, since browsers block local start pages for security reasons. I use New Tab Override, in case any Firefox user wants to try it.

Why create another start page?

Well, this is an important question because there are thousands of start pages out there — in fact, the browser itself comes with one. So why would I go and build my own? The first reason is that Firefox's shortcuts are terrible. I could go on about them, but I won't even bother.

Another reason is that I wanted to customize it at a technical level to streamline my day-to-day, and this is where it gets interesting — but I won't spoil it here, you can read about it in the design and code sections hehe. That said, I honestly think the quality of both texts is far below the quality of the other case studies, so I apologize in advance if you notice it too — though who knows, it might just be imposter syndrome.

Design

For the design I used a minimalist and brutalist approach, inspired by Linux terminals, along with a customization system that includes 5 themes and the ability to add custom side decorations.

Code

Honestly, I don't know what to say about the code without spoiling it, but basically it's almost the same thing I always do, with some borrowed code from Aerko_, and not much else really.
